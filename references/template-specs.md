# 三份交付文件字段规格说明

## 一、领队计划 (.docx)

### 1. 活动介绍
- **类型：** 叙述性文字 + 配图
- **内容：** 路线历史/文化背景、自然景观、季节特色、活动亮点
- **语气：** 文学化、有感染力
- **篇幅：** 2-4 段
- **注意：** 配图标注来源("图源自网络，侵删")；天气依赖型景观加免责声明

### 2. 路线信息
- **2.1 徒步路线：** 路线名称、起终点、总里程、累计爬升/下降、路线类型(环线/穿越)、分段描述
- **2.2 路线强度：** 数值分数 + 等级，附公式说明
- **2.3 路线难度与风险点：** 各特殊路况具体位置 + 应对建议
- **2.4 总体评价：** 一句话批准声明，引用安全管理条例

### 3. 组织事宜
| 字段 | 格式 |
|------|------|
| 活动时间 | YYYY年M月D日（周X） |
| 领队 | 姓名列表，空格分隔 |
| 招募人数 | 数字+单位(含领队说明) |
| 预收费用 | 会员X元，非会员+10元 |
| 费用包含 | 列表形式 |
| 费用不包含 | 列表形式 |
| 时间安排 | HH:MM——地点/事项，按时间顺序 |
| 集合地点 | 校门名称列表 |
| 交通方式 | 包车/公共交通 |

### 4. 物资准备
- 分类：必备 / 强烈推荐 / 可选
- 按类别：衣物、鞋袜、饮食、其他
- 标注协会可租借装备(登山杖5元/根)

### 5. 报名与注意事项
- 报名二维码(文字描述位置)
- 关键时间点(报名开始/退款截止/报名截止)
- 核心规则(不过向导不落押后/迟到不候等)

## 二、活动备案表 (.docx)

### 标准表头
| 字段 | 内容 |
|------|------|
| 单位名称 | 北京大学学生徒步越野协会 |
| 活动名称 | [路线名]一日徒步 / [路线名]轻装穿越 |
| 填表日期 | YYYY.M.D |
| 活动人数 | 数字 |
| 时间 | YYYY.M.D 或 YYYY.M.D-YYYY.M.D |
| 地点 | 省/市/区 |
| 是否有校外嘉宾 | 否 |

### 活动负责人表
列：活动负责人 | 院系/年级 | 邮箱 | 手机
- 每行一位领队
- 包含实习领队

### 正文 5 节
1. **线路简介：** 起终点、总里程、总爬升、分段路况、强度等级、适用人群
2. **领队介绍：** 总人数/领队数/比例/主领队资格人数 + 每人详细bio
3. **线路风险：** 风险预案表截图 + 一段话总结
4. **行进节奏：** 时间线，分段标注里程爬升
5. **装备与注意事项：** 衣物/鞋袜/饮食/必备物品

### 审批签字区
- 学生社团负责人意见(社长)
- 指导教师意见
- 社团挂靠单位意见
- 校团委意见

## 三、风险预案 (.xlsx)

### 标题行(合并A1:F1)
`YYYY年M月D日徒协[路线名]徒步风险分析和应对要求`

### 表头(第2行)
序号 | 路线地名 | 风险原因 | 风险级别 | 应对要求 | 备注

### 标准 8-9 行风险项

| 序号 | 风险原因 | 默认风险级别 |
|------|----------|-------------|
| 1 | 交通意外 | 安全 |
| 2 | 下雨、失温、中暑 | 视季节/天气而定 |
| 3 | 队员体能耗竭 | 风险存在 |
| 4 | 队伍间距过大走散 | 安全/风险存在 |
| 5 | 不能到达目的地 | 安全/风险容易发生 |
| 6 | 崴脚 | 风险存在 |
| 7 | 陡坡滑坠 | 安全/风险容易发生/很危险 |
| 8 | 动植物风险 | 风险存在 |
| 9 | 下撤路线/备选路线 | — |

### 风险级别色标(右侧 I-J 列)

| 级别 | 颜色 | 描述 |
|------|------|------|
| 安全 | 绿 | 没有环境风险 |
| 风险存在 | 黄 | 没有环境风险，但有不可预见的风险存在 |
| 风险容易发生 | 橙 | 环境风险较高，会发生高致伤率 |
| 危险 | 红 | 风险发生概率低但致伤后果严重 |
| 很危险 | 深红 | 风险发生概率高，发生后严重容易致残 |
| 极度危险 | 黑/深红 | 风险无法承受，概率高且容易致死 |

### 格式要求
- 标题行合并居中加粗
- 表头加粗有边框
- E列(应对要求)最宽，文本自动换行
- 风险级别列需填色
