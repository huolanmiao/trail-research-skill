# 北大徒协领队手册精要

> 摘录自领队手册.md (版本 2026.4.20)，用于生成符合协会规范的文档。

## 路线分级标准

**标准强度计算公式：**
```
路线强度 = 路程(km) + 爬升(m)/100
```

**分级阈值：**
| 等级 | 强度范围 |
|------|----------|
| 休闲 | < 12 |
| 初级 | 12 ~ 20 |
| 中级 | 20 ~ 35 |
| 高级 | 35 ~ 50 |
| 超高级 | > 50 |

**特殊路况调整规则：**
- 出现 1-2 个特殊路况 → 提升一个等级
- 重装 → 额外提升一个等级

**特殊路况清单：** 积雪、野长城、暴露感强、雨后烂泥、小陡崖、溯溪、踏冰、抱石

**路况分类（基础）：** 柏油路、公路、防火道、机耕路、木栈道、石板路、平整土路、狭窄土路、碎石路

## 领队人数要求（至少）

| 难度 | 37座车 | 50座车 |
|------|--------|--------|
| 初级 | 3正式+2实习 | 4正式+2实习 |
| 中级 | 3正式+2实习 | 4正式+2实习 |
| 高级 | 参照中级 | 参照中级 |

- 每次活动有且仅有一名**主领队**，其余为副领队
- 实习领队数量不超过 4 名

## 领队补贴

**单日活动：**
- 主领队（正式）300元/人
- 副领队（正式）150元/人
- 正式领队和实习领队均可退活动报名费
- 单次补贴总额不超过 750 元

**长线（含露营）：**
- 总补贴 500×(n+1)，n 为徒步天数
- 默认备案表挂名的正式领队均分

## 时间线要求

| 时间节点 | 事项 |
|----------|------|
| 活动前一周周一前 | 确定活动时间和正式领队 |
| 活动前一周周二前 | 提交备案表、领队计划，轨迹文件发领队组 |
| 活动前一周周三前 | 理事会审核备案表，转发备案群 |
| 活动当周周一 | 宣发推送、开始报名 |
| 活动当周周五中午 | 报名截止、确定名单 |
| 活动前一天 | 再次确定所有事宜，发布群公告 |

## 费用构成

1. **车费** — 提前询问外联部
2. **领队补贴** — 见上文
3. **免活动费** — 实习领队和领队退还的活动费
4. **飘带** — 5元/条
5. **保险** — 2元/人/天
6. **剩余** — 超过10元/人且超过报名费10%时退款

## 车辆规则

- 单日徒步最少 38 座车，少于 38 座视为私约
- 50/54/55 座价格相等
- 露营包车座位数 ≈ 活动人数×1.5
- 座位数包含司机

## 接驳顺序

- 去北京西边（门头沟等）：医学部 → 本部 → 万柳
- 去北京东边（密云、怀柔）：万柳 → 本部 → 医学部
- 医学部→本部：15分钟，本部→万柳：15分钟
- 医学部→新燕园：35分钟，万柳→新燕园：40分钟
- 预留 10-15 分钟上厕所时间

## 装备清单（基础）

**必备：** 防水防风外套/雨衣、保暖衣物(抓绒/薄羽绒)、速干衣裤、徒步鞋/越野跑鞋、1.5-2L水、午餐

**强烈推荐：** 手台、登山杖、护膝

**按季节：** 冬天冰爪/雪套/保温水壶；夏天雨衣/防晒；夜爬头灯；沙漠沙套

## 活动当天流程

1. 出发前点名 → 2. 车程中安排洗手间 → 3. 下车前确认司机汇合 → 4. 组织热身+登山杖教学 → 5. 合影 → 6. 行进中关注队伍 → 7. 返程清点人数报平安 → 8. 简要总结 → 9. 发送反馈问卷

## 四段论队伍管理

向导 - 前中段 - 后中段 - 押后，领队前后游走，手台沟通。
